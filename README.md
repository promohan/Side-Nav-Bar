# 🔐 SIde-Nar Bar Project

Welcome to the **SIde-Nar Bar Project** — a simple, clean, and responsive login interface built using HTML5, CSS3, and JavaScript.  
Live demo: [https://promohan.github.io/login_form/](https://promohan.github.io/Side-Nav-Bar/#)

---

## 🌟 Features    
- Responsive design: adapts to mobile, tablet, desktop  
- Optional theme toggle (Light / Dark mode) for better UX  
- Semantic HTML, modular CSS, and minimal JavaScript for toggling themes or basic form validation  
- Hosted easily via GitHub Pages  

---

## 🧰 Tech Stack

| Technology       | Purpose                          |
|------------------|----------------------------------|
| **HTML5**        | Structured form markup           |
| **CSS3**         | Styling, theme toggle, responsiveness |
| **JavaScript**   | Theme toggle logic and form interactivity |
| **GitHub Pages** | Free static hosting and live link |
